4、setup.S会为系统设置好必要的硬件参数，然后将0x10000处开始的系统模块代码重新拷贝至0x0内存处，接着设置好临时gdt和idt表、打开保护模式，最后跳往0x0处开始执行32位系统模块头部代码（即head.s）。
